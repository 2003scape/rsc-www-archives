Unable to start java applet

Either your browser doesn't support java or you have java disabled.

Troubleshooting

  • To play this game you need to have a java capable browser, such as Netscape 4 or Internet Explorer 4 or 5.
  • If the applet still doesn't appear you may have java disabled on your browser. Go to the options/preferences menu and ensure java is enabled.

Controls
Use the arrows keys to move your ship, and spacebar to fire the main weapon.
hold N or M to charge the plasma gun, release the key to fire.

Press R to change the screen resolution. If the game is running slowly the resolution
should decrease automatically, if it doesn't or you want extra speed just press R.

Hints
The plasma gun can shoot through walls! Use it to get those annoying gun turrets.
Look for safe routes through the level, some routes are easier than they look
