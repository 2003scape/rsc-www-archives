Unable to start java applet

Either your browser doesn't support java or you have java disabled.

Troubleshooting

  • To play this game you need to have a java capable browser, such as Netscape 4 or Internet Explorer 4 or 5.
  • If the applet still doesn't appear you may have java disabled on your browser. Go to the options/preferences menu and ensure java is enabled.

Instructions
To win each level you must paint all the blue-blocks purple using the paint-pots scattered around. There are numerous blocks which will help or hinder your progress. Use the tutorial in the game for more information about each block.

Controls
Use the arrows key to move, and return to jump.
Press f1 to use dynamite, and f2 to use a jetpack.
Use the < and > keys to rotate the camera.
