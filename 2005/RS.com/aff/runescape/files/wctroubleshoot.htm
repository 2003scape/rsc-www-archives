Windows Client Troubleshooting

If you having problems with the Windows Client try following these steps in order and see if you can resolve your problem.

1. When the Windows Client loads, I get a page saying 'Page cannot be displayed'

This is most likely to do with a personal firewall which is blocking the Windows Client from accessing the internet. You will have to allow the program "runescape.exe" access to the internet in order for it to work.

2. I don't get a scrollbar

This is because you are running an old version of Internet Explorer. You should visit Microsoft Windows Update to download the latest version.

3. The website appears but the game itself won't work

This is because you have been using Sun Java to play RuneScape but have a buggy version of Microsoft Java installed. RuneScape will work in the Windows Client if you do one of these two things:
  1. On the detail select page always select "Signed Applet Using Sun Java" from the drop down options before selecting the appropriate detail option.
  2. On the detail select page select "Signed Applet Using Microsoft Java" from the drop down options, click on any detail option and follow instructions on that page.
If you choose the second option you will only have to do this once and will most likely see a performance benefit because Microsoft Java is slightly better at playing RuneScape than Sun Java.
